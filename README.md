# New York City-Traffic-Accidents

Link to the dataset: https://mavenanalytics.io/data-playground/nyc-traffic-accidents

original dataset: https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Crashes/h9gi-nx95/about_data


- "ON STREET NAME" means "Street on which the collision occurred".
- "CROSS STREET NAME" means "Nearest cross street to the collision".
- "OFF STREET NAME" means "Street address" if known.
